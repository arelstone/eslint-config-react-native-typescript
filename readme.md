## Usage

```
yarn add @arelstone/eslint-config-react-native-typescript --dev
// or
npm install -D @arelstone/eslint-config-react-native-typescript
```

In `.eslintrc`:

```json
{ 
  "extends": "@arelstone/eslint-config-react-native-typescript", 
} 
```
